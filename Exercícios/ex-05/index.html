Formatações

////

Negrito

para fazer um texto com negrito use a tag STRONG

////

Itálico

para usar o Italico utilize a tag EM

////

marcação

para fazer um texto marcado como se fosse corretivo utilize a tag MARK

////

grande e pequeno

para fazer um texto grande utilize a tag BIG (obsoleto)

e para um texto menor utilize a tag SMALL

////

deletado/inserido (razurado/sublinado)

para fazer um texto deletado(razurado) utilize a tag DEL

e para fazer um texto Inserido (sublinhado) utilize a tag INS

////

sobrescrito e subscrito

para fazer um texto sobrescrito (acima dos outros digitos) utilize a tag SUP

para fazer um texto subscrito (abaixo dos outros digitos) utilize a Tag SUB
